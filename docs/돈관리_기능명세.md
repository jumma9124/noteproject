│   예금 15%   │  □ 현금: 500만원                   │

### 기능
- 자산 항목 클릭 → 해당 자산 세부 페이지
- 원형 차트: 자산 + 부채(빨간색) 비율 표시
- 실시간 현황 한눈에 파악

## 2. 주식 세부 페이지

### 종목 목록 (테이블)
```
┌──────────────────────────────────────────────────┐
│ ← 뒤로  │  주식 상세                                │
├──────────────────────────────────────────────────┤
│ 총 투자금: 2,100만원  │  현재 평가액: 2,450만원    │
│ 총 수익률: +16.7% (+350만원) 🟢                   │
├──────────────────────────────────────────────────┤
│ 보유 종목                           [+ 새 종목 추가]│
│                                                  │
│ ┌────────┬────────┬────────┬────────┬────────┐ │
│ │ 종목명  │ 평단가  │ 현재가  │ 수익률  │ 상세   │ │
│ ├────────┼────────┼────────┼────────┼────────┤ │
│ │삼성전자│ 65,000 │ 72,000 │+10.8%🟢│ [보기] │ │
│ │        │        │ (2/25) │        │        │ │
│ │카카오  │ 45,000 │ 48,000 │ +6.7%🟢│ [보기] │ │
│ │현대차  │120,000 │115,000 │ -4.2%🔴│ [보기] │ │
│ └────────┴────────┴────────┴────────┴────────┘ │
└──────────────────────────────────────────────────┘
```

### 종목 상세 (매수 내역)
```
┌──────────────────────────────────────────────────┐
│ 삼성전자 매수 내역                   [+ 매수 추가]  │
│ ┌────────────┬──────┬──────────┬──────────────┐ │
│ │ 매수일      │ 수량 │ 매수가    │ 금액         │ │
│ ├────────────┼──────┼──────────┼──────────────┤ │
│ │ 2026-02-01 │ 10주 │ 60,000원 │ 600,000원    │ │
│ │ 2026-02-03 │ 15주 │ 68,000원 │ 1,020,000원  │ │
│ ├────────────┴──────┴──────────┼──────────────┤ │
│ │ 평균 매수가 (자동계산)          │ 65,000원     │ │
│ │ 총 보유                       │ 25주         │ │
│ └───────────────────────────────┴──────────────┘ │
│                                                  │
│ 현재가 업데이트: [72,000원] [저장]                │
└──────────────────────────────────────────────────┘
```

### 기능
- 평단가 자동 계산
- 현재가 **수동 입력** (저녁 루틴)
- 수익률 자동 계산
- 매수/매도 내역 관리

## 8. 주별 지출 입력 페이지

### 레이아웃
```
┌─────────────────────────────────────┐
│ 💸 2026년 2월 - 3주차 지출 입력       │
├─────────────────────────────────────┤
│ 이번 주 예산: 70만원                 │
│ 현재 지출: 45만원                    │
│ 남은 예산: 25만원 ✅                 │
│                                    │
├─────────────────────────────────────┤
│ 지출 내역                            │
│ ┌────────┬────────────┬──────────┐ │
│ │ 날짜    │ 메모(선택)  │ 금액     │ │
│ ├────────┼────────────┼──────────┤ │
│ │ 2/17(월)│ 점심/커피   │  20,000  │ │
│ │ 2/18(화)│ 교통/식비   │  15,000  │ │
│ │ 2/19(수)│            │  10,000  │ │
│ └────────┴────────────┴──────────┘ │
│                                    │
│ [+ 지출 추가]                        │
│                                    │
│ ───────────────────────────────     │
│ 주간 총 지출: 45만원                 │
└─────────────────────────────────────┘
```

### 특징
- **카테고리 없이** 간단하게
- 메모는 선택사항
- 주별로 자동 집계
- 고정비는 제외된 변동비만

## 12. 예산 시스템 구조

### 연 예산
```
연 예산: 5,000만원
├─ 고정비 (자동 차감)
│   ├─ 월세: 100만원 × 12 = 1,200만원
│   ├─ 통신비: 10만원 × 12 = 120만원
│   └─ 기타 고정비
│
└─ 변동비 (실제 관리 대상)
    ├─ 1월 예산: 300만원
    │   ├─ 1주차: 80만원 (사용: 65만원)
    │   ├─ 2주차: 70만원 (사용: 82만원 ⚠️)
    │   └─ ...
    ├─ 2월 예산: 280만원
    └─ ...
```

## 14. 주요 특징

- ✅ 카테고리 없는 간단한 지출 관리 (주별 묶음)

- ✅ 원형 차트 시각화
- ✅ 목표 대비 달성률 표시
