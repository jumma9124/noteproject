# 다이어리 기능 명세서

## 1. 메인 대시보드

### 레이아웃
```
┌──────────────────┬──────────────────────────┐
│                  │ 총연차: 15  사용: 3  남음: 12 │
│                  ├──────────────────────────┤
│     달력         │ [회사 할일] │ [개인 할일]  │
│   2026년 2월     │ D-2 □ 보고서│ D-1 □ 운동  │
│                  │ D-5 □ 미팅  │ D-3 □ 예약  │
│  일 월 화 수 목 금 토│ (미완료만)  │ (미완료만)   │
│                  │             │            │
│ ┌──┐ ┌──┐       │             │            │
│ │1🔴│ │2 │ 3🔴🔵 │             │            │
│ │▓▓│ └──┘       │             │            │
│ └──┘             │             │            │
└──────────────────┴──────────────────────────┘
```

### 기능
- **읽기 전용** (수정 불가)
- 달력 월 이동 (◀ 2026년 2월 ▶)
- 날짜 클릭 → 상세 페이지 이동

### 달력 표시 요소

#### 점 표시
- 🔴 **빨간 점**: 해당 날짜가 마감일인 할일 있음 (완료 여부 무관, 기록용)
- 🔵 **파란 점**: 그날 일기 작성함
- 🔴🔵 **둘 다**: 마감 할일도 있고 일기도 씀
- 날짜 오른쪽에 표시

#### 휴가 표시 (주황색 테두리)
날짜 칸을 위에서부터 세로로 4등분:
- **연차(8h)**: 날짜 칸 전체 주황 테두리
- **반차(4h)**: 위 절반 영역만 주황 테두리
- **반반차(2h)**: 위 1/4 영역만 주황 테두리

### 연차 정보
- **총연차**: 설정에서 수동 입력
- **사용연차**: 달력에서 날짜마다 입력 → 자동 합산
  - 반반차 = +0.25일
  - 반차 = +0.5일
  - 연차 = +1일
- **남은연차**: 자동 계산 (총연차 - 사용연차)

### 할일 체크리스트
- **회사 / 개인** 분리 표시
- **D-day 자동 계산** (마감일 기준)
- **급한 순 자동 정렬** (D-day 작은 순)
- **미완료 항목만** 표시
- 긴급 표시된 항목 = 빨간 테두리

## 2. 날짜 상세 페이지

### 레이아웃
```
┌─────────────────────────────────────┐
│ ← 뒤로  │  2026년 2월 3일            │
├─────────────────────────────────────┤
│ 🏖️ 휴가 설정                         │
│ ○ 없음  ○ 반반차(2h)  ○ 반차(4h)     │
│ ○ 연차(8h)                          │
│                                    │
├─────────────────────────────────────┤
│ [+ 새 할일 추가]                     │
│                                    │
│ 📋 오늘의 할일                       │
│ 회사:                               │
│ D-2 □ 보고서 작성 [긴급]             │
│ D-5 ☑ 회의 준비 (완료, 취소선)        │
│                                    │
│ 개인:                               │
│ D-1 □ 운동하기                      │
│                                    │
├─────────────────────────────────────┤
│ 📝 오늘의 일기                       │
│                                    │
│ (마크다운 에디터)                    │
│ 오늘 프로젝트 회의했는데...          │
│ 저녁에 안세영 경기 봤음!             │
│                                    │
│ [자동저장됨]                        │
└─────────────────────────────────────┘
```

### 기능
- **모든 수정 가능** (Edit)
- 휴가 설정/변경
- 할일 추가/수정/완료 체크/긴급 표시
- 일기 작성/수정 (마크다운)

### 할일 입력 정보
- 할일 내용
- 마감일
- 카테고리 (회사/개인)
- 긴급 여부 (체크박스)

### 일기
- 해당 날짜 상세 페이지에서만 보임 (완전 프라이빗)
- 마크다운 에디터
- 자동 저장

## 3. 전체 할일 페이지

### 레이아웃
```
┌─────────────────────────────────────┐
│ 전체 할일 목록                       │
├─────────────────────────────────────┤
│ [회사] [개인] [완료됨] ← 탭          │
│                                    │
│ D-2 □ 보고서 작성 (2/3 등록)         │
│ D-5 ☑ 회의 준비 (2/1 등록)          │
│ D-8 □ 자료 제출 (1/28 등록)         │
│                                    │
│ [수정] [삭제]                       │
└─────────────────────────────────────┘
```

### 기능
- **모든 수정/삭제 가능**
- 삭제하지 않은 모든 할일 표시
- 완료/미완료 탭 분리
- 회사/개인 필터

## 6. 기술 스택

- **Frontend**: React (PC 전용, 모바일 제외)
- **Backend/DB**: Firebase
  - Firestore (데이터베이스)
  - Authentication (로그인)
  - Hosting (배포)
- **배포**: GitHub Pages 또는 Firebase Hosting
- **에디터**: 마크다운 (코드 하이라이팅 지원)

## 7. 주요 특징

- ✅ PC 중심, 모바일 제외
- ✅ 손글씨 지원 X
- ✅ 마크다운 기반
- ✅ 개인/회사 구분 (회사는 간단한 것만)
- ✅ 과거/미래 데이터 관리 (2~3년)
- ✅ 로그인/회원가입 (Firebase Auth)
- ✅ 메인은 읽기 전용, 상세 페이지에서만 수정
